# Приложение озвучки текста с использованием Silero TTS

Это приложение предоставляет графический интерфейс для преобразования текста в речь с использованием модели [Silero TTS v5](https://github.com/snakers4/silero-models).

## Функциональность

- **Озвучка текста на русском языке** с использованием модели Silero TTS v5
- **Выбор голоса**: доступны голоса `baya`, `eugene`, `kseniya`, `xenia`, `random`
- **Настройка скорости речи**: x-slow, slow, medium, fast, x-fast
- **Предобработка текста**:
  - Преобразование чисел в слова (123 → сто двадцать три)
  - Автоматическая расстановка ударений (silero-stress)
  - Дополнительная обработка через ruaccent (опционально)
  - Замена сокращений (ул. → улица, г. → город)
- **Работа с длинными текстами**: автоматическое разбиение на чанки с настраиваемыми параметрами
- **Сохранение аудио**: в форматах WAV и MP3 с настраиваемым битрейтом
- **Объединение аудио**: слияние нескольких WAV файлов в один MP3
- **Загрузка файлов**: поддержка форматов TXT, FB2, ZIP (с FB2 внутри)
- **Целевая директория**: настройка папки для сохранения аудиофайлов
- **Протокол работы**: логирование всех операций в отдельной вкладке

## Установка

1. Клонируйте или скачайте проект

2. Создайте виртуальное окружение (опционально):


## Использование

### Запуск GUI интерфейса

```bash
python text2mp3.py
```

### Командная строка (CLI)

Приложение поддерживает работу из командной строки для автоматизации и пакетной обработки:

```bash
# Озвучка текста с сохранением в WAV
python text2mp3.py --text "Привет мир!" -o output.wav

# Озвучка из файла с конвертацией в MP3
python text2mp3.py -i input.txt -o output.mp3 --mp3

# Режим чанков для длинных текстов
python text2mp3.py --text "Длинный текст..." --chunks --max-chars 1000

# С предобработкой и расстановкой ударений
python text2mp3.py -i book.txt --preprocess --ruaccent -o audiobook.mp3 --mp3

# Выбор голоса и скорости
python text2mp3.py --text "Текст" -s kseniya -r slow -o slow.wav

# Подробный режим логирования
python text2mp3.py -i text.txt -o result.mp3 --mp3 -v
```

#### Параметры CLI:

| Параметр | Краткий | Описание |
|----------|---------|----------|
| `--text` | `-t` | Текст для озвучки |
| `--input-file` | `-i` | Файл с текстом (txt, fb2) |
| `--output` | `-o` | Выходной файл (WAV или MP3) |
| `--mp3` | | Конвертировать в MP3 |
| `--bitrate` | | Битрейт MP3 (128k, 192k, 256k, 320k), по умолчанию 192k |
| `--no-wav` | | Удалить WAV после конвертации в MP3 |
| `--speaker` | `-s` | Голос диктора (baya, eugene, kseniya, xenia, random) |
| `--speech-rate` | `-r` | Скорость речи (x-slow, slow, medium, fast, x-fast) |
| `--threads` | | Количество потоков CPU, по умолчанию 4 |
| `--chunks` | | Разбивать текст на чанки |
| `--max-chars` | | Макс. символов в чанке, по умолчанию 1200 |
| `--silence-ms` | | Пауза между чанками (мс), по умолчанию 200 |
| `--save-parts` | | Сохранять чанки в отдельную директорию |
| `--delete-parts` | | Удалить директорию с частями после конвертации в MP3 |
| `--output-dir` | | Целевая директория для сохранения аудио |
| `--preprocess` | | Использовать предобработку текста |
| `--no-num2words` | | Не заменять числа словами |
| `--ruaccent` | | Использовать ruaccent для ударений |
| `--verbose` | `-v` | Подробный вывод логов |
| `--quiet` | `-q` | Не выводить логи (только ошибки) |
| `--gui` | | Запуск GUI интерфейса |

2. **Основные операции в GUI:**
   - Введите текст или загрузите из файла (TXT/FB2)
   - Выберите голос и скорость озвучки
   - Нажмите **"▶ Сгенерировать и воспроизвести"** для прослушивания
   - Нажмите **"💾 Сохранить (WAV/MP3)"** для сохранения в файл

3. **Работа с длинными текстами:**
   - Тексты длиннее 3000 символов автоматически разбиваются на чанки
   - Настройка параметров разбиения во вкладке "Кусочки"
   - Возможность ручного разбиения текста кнопкой **"✂ Разделить текст на кусочки"**

4. **Настройка целевой директории:**
   - Укажите папку для сохранения аудиофайлов в поле "Целевая директория"
   - Кнопка **"📁"** открывает диалог выбора папки
   - Кнопка **"📁 Открыть папку аудио"** открывает папку в проводнике

5. **Объединение в MP3:**
   - Кнопка **"🔗 Объединить в MP3"** сливает все WAV файлы из целевой директории
   - Битрейт MP3 настраивается в параметрах (по умолчанию 192k)

## Настройки

Приложение автоматически сохраняет настройки в файл `tts_config.json`:
- Выбранный голос и скорость
- Последний использованный текст
- Параметры разбиения на чанки
- Целевая директория для аудио
- Настройки MP3 (битрейт)
- Геометрия окна
- Настройки предобработки текста

## Предобработка текста

Приложение поддерживает предобработку текста для улучшения качества синтеза речи:

### Возможности:

1. **Преобразование чисел в слова**
   - `123` → `сто двадцать три`
   - `3.14` → `три целых четырнадцать сотых`
   - `1 000 000` → `один миллион`

2. **Расстановка ударений**
   - Используется модель `silero-stress`
   - Формат: `+` перед ударной гласной (например, `з+амок`)
   - Опционально: `ruaccent` для повышенной точности

3. **Замена сокращений**
   - `ул.` → `улица`
   - `г.` → `город`
   - `д.` → `дом`
   - `млн.` → `миллион`
   - и более 50 других сокращений

### Примеры использования:

**Текст до предобработки:**
```
Я живу на ул. Ленина, д. 25, кв. 123. Мой номер 8-900-555-35-35.
Встреча в 14.30 у г. Москвы.
```

**Текст после предобработки:**
```
Я живу на у́лица Ле́нина, до́м 25, квартира́ сто два́дцать три. Мой но́мер во́семь-девятьсо́т-пятьсо́т пять́десят три-пять́десят пять-три́дцать пять.
Встре́ча в четы́рнадцать часо́в три́дцать мину́т у го́род Москвы́.
```

### Настройка:

Во вкладке "Предобработка текста" доступны опции:
- ☑ Использовать предобработку текста
- ☑ Заменять числа словами (123 → сто двадцать три)
- ☐ Доп. обработка (ruaccent, медленнее)

**Примечание:** При первом включении предобработки модели загружаются автоматически (может занять несколько секунд).

## Расположение файлов

- **Модель:** `%LOCALAPPDATA%\SileroTTS\models\v5_ru.pt`
- **Аудио по умолчанию:** `%LOCALAPPDATA%\SileroTTS\audiofiles\`
- **Настройки:** `tts_config.json` (в папке приложения)
