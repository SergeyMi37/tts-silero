# Приложение озвучки текста с использованием Silero TTS

Это приложение предоставляет графический интерфейс для преобразования текста в речь с использованием библиотеки [silero-tts-enhanced](https://github.com/daswer123/silero-tts-enhanced).

## Функциональность

- Преобразование текста в речь на нескольких языках (русский, английский, немецкий, испанский)
- Настройка скорости речи
- Поддержка нескольких языков
- Простой и интуитивно понятный интерфейс

## Установка

1. Клонируйте или скачайте проект
2. Установите зависимости:

Create virtual environment (optional)
``` bash
python3 -m venv env-lin
source env-lin/bin/activate

# Create virtual environment for Windows
python -m venv env-win
source env-win/Scripts/activate

```bash
pip install -r requirements.txt
```

## Использование

1. Запустите приложение:

```bash
python tts_app.py
```

2. Выберите язык из выпадающего списка
3. Введите текст в текстовое поле
4. Нажмите кнопку "Озвучить" для преобразования текста в речь
5. Используйте ползунок для регулировки скорости речи

## Зависимости

- Python 3.7+
- torch
- numpy
- requests
- silero-tts-enhanced
- pygame

## Особенности реализации

- Приложение использует библиотеку Silero TTS для преобразования текста в речь
- Для воспроизведения аудио используется pygame как основной метод, с fallback к системным средствам
- Модели загружаются автоматически при первом запуске
- Интерфейс реализован с использованием tkinter

## Возможные проблемы

- При первом запуске может потребоваться время для загрузки моделей
- Для корректного отображения и обработки кириллического текста могут потребоваться дополнительные настройки кодировки в зависимости от системы
